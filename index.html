My Projects

Audio/Text to sign language

The project aims to develop a website that converts English text into Indian Sign Language (ISL) to facilitate communication between hearing-impaired individuals and others. The system uses audio input, converts it to text, and then uses natural language processing (NLP) to interpret and choose the appropriate ISL signs. The software is written in Python and uses libraries such as Django and NLP, and the hardware required includes a device with a microphone, monitor, and internet connectivity.

Loan Prediction using Machine Learning Algorithm

A machine learning model was developed to predict loan approval scenarios based on various financial and demographic factors. This project involved using Python along with popular libraries such as pandas, NumPy and scikit-learn. The process included comprehensive data preprocessing to clean and prepare the data, exploratory data analysis to uncover insights, and the training and evaluation of multiple classifiers to achieve accurate predictions.

HandDino – Gesture-Controlled Chrome Dino Game using OpenCV

Developed a computer vision-based controller to play the Chrome Dino game using real-time hand gestures detected via webcam.
Used OpenCV to track motion and implemented gesture-based “jump” functionality with PyAutoGUI to simulate spacebar presses.
Tuned HSV color thresholds to handle varying lighting conditions, improving gesture accuracy.
